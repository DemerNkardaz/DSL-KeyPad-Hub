<ArticleSection title="События">

События реализуют возможность реагировать на определённые действия в программе. Их можно использовать для изменения поведения и наполнения программы. Например, добавление новых символов на стадии их инициализации, редактирование стандартной библиотеки символов, добавление новых режимов в «Альтернативный ввод» и так далее.

<ArticleSection is-sub-section title="Использование событий">

Следующая функция позволяет подписаться на событие:

```ts
Event.OnEvent(eventClass: <string>, eventName: <string>, callback: <function>)
```
<StyledText text="eventClass" preset="var"/> <StyledText text="string" preset="typ"/> — класс события, например, <StyledText text="Application" preset="qstr"/> или <StyledText text="Character Library" preset="qstr"/><br />
<StyledText text="eventName" preset="var"/> <StyledText text="string" preset="typ"/> — имя события внутри класса, например, <StyledText text="Initialized" preset="qstr"/> или <StyledText text="Raw Entries Counted" preset="qstr"/><br />
<StyledText text="callback" preset="var"/> <StyledText text="function" preset="typ"/> — функция, которая будет вызвана при наступлении события. Она может принимать аргументы, которые зависят от конкретного события.

<br />

#### Пример использования события

Добавление новых символов в библиотеку символов:
```rust
local characters := JSON.LoadFile("path/to/data.json", "UTF-8")

Event.OnEvent("Character Library", "Raw Entries Counted", () => ChrReg(characters))
```

<StyledText text="ChrReg" preset="cls"/> — класс, регистрирующий новые символы в библиотеке символов.<br />
Принимает 3 аргумента:

<StyledText text="rawEntries" preset="var"/> <StyledText text="array" preset="typ"/> `[entryName, { entryData } ... ]` массив пар<br />
<StyledText text="typeOfInit" preset="var"/> <StyledText text="string" preset="typ"/> `Internal` | `Custom` тип инициализации <br />
<StyledText text="preventProgressGUI" preset="var"/> <StyledText text="bool" preset="typ"/> `True` | `False` отключение | включение полосы прогресса

<br />

Модификация существующих символов в библиотеке символов:
```lua
local characters := JSON.LoadFile("path/to/data.json", "UTF-8")

Event.OnEvent("Character Library", "Raw Entries Counted", Handle)

Handle(&registrar, &rawEntries, &nameToID) {
  if charactersModifyData is Array && charactersModifyData.Length >= 2 {
    Loop charactersModifyData.Length // 2 {
      local index := A_Index * 2 - 1
      local entryName := charactersModifyData[index]

      if registrar.IsRawEntryExists(&entryName) {
        local entry := charactersModifyData[index + 1]
        registrar.ModifyRawEntry(&entryName, &rawEntries, &entry)
      }
    }
  }
}
```

Событие <StyledText text="Raw Entries Counted" preset="qstr"/> класса <StyledText text="Character Library" preset="qstr"/> позволяет получить доступ к необработанным данным символов и передаёт следующие ссылки в callback-функцию:

<StyledText text="this" preset="varRef"/> <StyledText text="class" preset="typ"/> `ChrReg { ... }` текущий экземпляр класса-регистратора символов<br />
<StyledText text="rawEntries" preset="varRef"/> <StyledText text="array" preset="typ"/> `[entryName, { entryData } ... ]` массив пар<br />
<StyledText text="nameToID" preset="varRef"/> <StyledText text="map" preset="typ"/> `Map(entryName, index, ... )` карта имён записей символов и их индексов<br />
